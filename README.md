# Excel::Transpose

## Usage

### Requirements

- Docker
- make

### 1. docker build

```
$ make docker_build
```

### 2. docker run

```
$ make docker_run EXCEL=/path/to/excel/file
```

### others

you can use tty on container

```
$ make docker_tty
```
